## 🎨 CUSTOMIZATION

### **To Add Real Images:**

Replace the gradient placeholders with real images:

```jsx
// In Home.jsx, replace:
<div className="image-placeholder restaurant-bg">

// With:
<img src="/images/restaurant.jpg" alt="Restaurant" />
```

### **To Change Colors:**

Edit `Home.css`:
```css
/* Change restaurant gradient */
.restaurant-bg {
  background: linear-gradient(135deg, #yourcolor1, #yourcolor2);
}
```

### **To Update Prices:**

Edit the room prices in `Home.jsx`:
```jsx
<p className="room-price">500 Birr/night</p>
```
