## 🎯 HOW TO USE

### **For Guests:**

1. **Go to Home Page**
   ```
   http://localhost:3000
   ```

2. **Click "Book Now" Button**
   - On the hotel feature card
   - Or go directly to: http://localhost:3000/rooms

3. **Browse Available Rooms**
   - See all 10 rooms
   - Check prices and features
   - View availability status

4. **Book a Room**
   - Click "📅 Book Now" on available room
   - Fill in booking form:
     - Guest name
     - Phone number
     - Check-in date
     - Check-out date
     - Special requests (optional)
   - See automatic price calculation
   - Click "Confirm Booking"

5. **Success!**
   - See confirmation message
   - Booking submitted
